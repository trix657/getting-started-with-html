Important Concepts

How the Web Works.

The web works by having a browser on a personal computer send a HTTP request for a webpage to a server. The server then finds the correct HTML(Hypertext Markup Language) document and returns that to the web browser which translates the code into what you see.

Tags and Elements

HTML documents are made up of Tags and Elements. Tags are components of HTML documents that specify the behavore of the text. Elements can be either Inline elements that directly effect that line, or Block elements that effect a block of code that with which the size of the block can be changed to fit your needs.
